# 学术 LaTeX 模板合集

整理常见学术期刊与会议的 LaTeX 模板，方便快速上手与共享。当前以中文说明为主，后续可按需补充英文或更多模板。

## 仓库结构

```text
.
├── README.md
├── journals/             # 期刊模板
│   ├── IEEE-Transactions/
│   ├── Elsevier-Neurocomputing/
│   ├── 计算机学报/
│   └── 控制理论与应用/
└── conferences/          # 会议模板
    └── AAMAS-2024/
```

## Release 下载

- 访问 [GitHub Releases](https://github.com/Ronchy2000/Academic-LaTeX/releases) 可直接获取最新打包模板，无需下载整个仓库。
- 工作流 `Package Templates`（推送 `v*` 标签或手动触发）会自动打包 `conferences/AAMAS-2024` 与 `journals` 下的模板，并生成对应的中英文说明。
- 压缩包文件名与内部目录统一使用 UTF-8 编码，包含中文模板名称（如 `控制理论与应用.zip`）时也能正确显示。
- 如果新增模板目录，请同步更新工作流中的打包列表，以便在发布时一并生成压缩包。

## 使用方式

1. 选择对应目录，阅读模板下的说明（`README.md` 或原始官方文档）。
2. 将模板复制到个人项目中，再进行论文撰写与版本管理。
3. 若模板内包含 `.cls`、`.sty` 或 `.bst` 文件，确保与 LaTeX 编译环境位于同一路径或正确引用。



## 模板维护规范

- **命名规则**：目录名称尽量使用“出版社/会议简称-年份”或“中文官方名称”，保持清晰易懂。
- **结构要求**：每个模板目录内应包含：
  - 官方模板源文件；
  - 示例 `tex` 文件；
  - 一份中文说明（如无官方中文资料，可自行撰写简要说明）。
- **更新建议**：新增模板时，在根目录 `README.md` 中同步更新目录结构，并在模板目录添加 `README.md` 说明来源与使用方法。

## 贡献方式

欢迎提交 PR：

1. 遵循上述目录与命名规范；
2. 标注模板来源（官网链接或原始压缩包）；
3. 给出最少编译说明（所需宏包、编译链如 `xelatex`/`pdflatex`）。

如遇版权或分发限制的模板，请先确认可公开分享后再提交。
