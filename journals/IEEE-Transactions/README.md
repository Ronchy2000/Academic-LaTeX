# IEEE Transactions LaTeX 模板

本目录收录 IEEE Transactions 期刊系列官方 LaTeX 模板，来源于 `IEEEtran` 官方示例。

## 文件说明

- `IEEEtran.cls`：IEEE 官方提供的文档类文件；
- `bare_jrnl_new_sample4.tex`：示例稿件，展示常用宏包与排版格式；
- `New_IEEEtran_how-to.tex/.pdf`：使用说明文档；
- `fig1.png`：示例插图。

## 快速使用

1. 将 `IEEEtran.cls` 与示例 `tex` 文件复制到个人项目中；
2. 根据示例修改稿件内容，可选择 `pdflatex` 或 `xelatex` 编译；
3. 提交前参考 `New_IEEEtran_how-to.pdf` 校对格式要求。

如需提交其他 IEEE 期刊模板，可在 `journals/` 下新增目录并补充中文说明。
