# 《控制理论与应用》期刊模板

本目录包含《控制理论与应用》杂志社发布的 LaTeX 模板压缩包，提供中文与英文两个版本。

## 文件说明

- `kzllyy template20181214.zip`：中文稿件模板；
- `kzllyy template_english20181219.zip`：英文投稿模板。

## 使用步骤

1. 解压对应的 `.zip` 文件，可放置在单独的工作目录中；
2. 阅读压缩包内的 `README` 或说明文档，按照杂志社要求撰写稿件；
3. 选择 `xelatex` 或杂志社推荐的编译方式进行测试；
4. 投稿前核对最新的投稿指南，确保模板仍符合官方要求。

> 模板来源于期刊官网，如有更新请及时替换压缩包并记录来源。
